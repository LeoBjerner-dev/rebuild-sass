TIPS FRÅN CHATGPT:

🔴 STRUCTURE (störst impact – gör först)
Ta bort “info overload” i headern (adress, mail, tider)
Gör en ren navbar:
logo vänster
3–4 länkar
CTA-knapp “Beställ”
Bygg om layout till:
Hero
Meny (cards)
Om oss
Kontakt/footer

🟡 HERO SECTION (det viktigaste visuellt)
Lägg mörk overlay på bilden
Centrera allt innehåll
Kortare text (max 1–2 rader)
EN tydlig CTA (inte 3 knappar)
Gör rubriken större och tydligare

🟢 NAVBAR
Lägg till hover-effekter
Öka spacing mellan länkar
Gör den sticky (stannar när man scrollar)
Ta bort tunga bakgrunder → mer clean

🔵 TYPOGRAFI
Byt font (Inter / Poppins)
Tydlig hierarki:
h1 stor
h2 medium
p mindre
Öka line-height (luftigare text)

🟣 BUTTONS
Gör större (padding!)
Rundade hörn
Hover-effekt (lyft upp / ändra färg)
Konsistent färg (inte olika orange nyanser)

⚫ SPACING / LAYOUT
Lägg till mycket mer luft:
padding: 60–100px per section
Centrera content (max-width)
Använd grid/flex istället för “hacky positioning”

🔥 MENY (största visuella upgrade)
Gör om till cards istället för lista
Lägg till:
bild
titel
pris
Grid layout (responsiv)

🟠 FÄRGER
Definiera 2–3 färger max
Använd samma orange överallt
Lägg till neutral bakgrund (#f9f9f9)

🟤 FOOTER
Flytta hit:
adress
öppettider
kontakt
Gör enkel 2–3 kolumn layout

📱 RESPONSIVE (många failar här)
Se till att:
navbar funkar på mobil
cards staplas snyggt
text inte blir för stor/liten
Lägg breakpoint vid ~768px

💣 EXTRA (det som gör att du sticker ut)
Hover på cards (lite “lift”)
Smooth scroll
Subtila animationer
Active state i navbar
