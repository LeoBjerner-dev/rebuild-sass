Ändringar jag gjort:

1. På deras hemsida så hade de bara en container med text och knappar som "hero section". Jag ändrade så att det istället visar deras toppsäljare direkt så att användaren snabbare ska förstå vad som erbjuds.

2. Jag gjorde nav tydligare genom att lyfta fram viktiga actions som att beställa och se meny direkt i headern. Originalet var mindre tydligt och krävde mer av användaren.

3. Jag tog bort onödigt innehåll som kartan och flyttade kontaktuppgifter till en separat "kontakt"flik.

4. Jag skapade en tydlig produktsektion med tre kort (cards) för toppsäljare och förbättrade spacing och layout så att sidan blir mer lättläst och strukturerad.
